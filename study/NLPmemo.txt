{{{ Thu Nov 14 22:20:06 JST 2013

下の論文についてもう一度ちゃんと読む
これをまとめて明日発表する

"Exploiting emoticons in sentiment analysis."
 Hogenboom, Alexander, et al.
Proceedings of the 28th Annual ACM Symposium on Applied Computing. ACM, 2013.
http://eprints.eemcs.utwente.nl/23268/01/sac13-senticon.pdf

# 目的

どんな顔文字がどんな感情を表すかの解析をしたい
顔文字，単語についてアノテーションした lexicon (辞書?) を
予め用意する

# 性質

* 顔文字の場所
84.0% の顔文字は段落の最後
9.0% の顔文字は中間くらい
7.0% が段落の最初
また中間に現れる顔文字は文と文をつなぐ (associate) ように使われる

仮定
emoticons have local effect
段落が複数の顔文字を含む場合には文に作用する
一つなら段落全体に対して affect すると仮定する
作用する対象を text segment という
segment は文または段落である

* 使われ方

emoticons are used in 3 types

例．一般的に次の顔文字は次のような意味だろう
  (^^) ... positive
  -_-  ... negative

1. テキストに感情が not clearly で，顔文字でexpress する
   (^^) I got a promotion
2. stress sentiment
   I love my work (^^)
3. negation
   I love my work -_-

# 解析

* step 1
split document -> segments (paragraphs or text)

* step 2
check that each segment has emoticons (or not)

use an emoticon sentiment lexicon
a list of `character sequence with sentiment scores`

* step 3a
辞書を用いて，顔文字から感情スコアを出す
顔文字eに対してスコアを
  sent(e)
と書く

* step 3b
辞書を用いて，語から感情スコアを出す
語wに対してスコアを
  sent(w)
と書く．但しここで語は，sentiment-carrying word もしくは形容詞に
限定する

ここで2種類の感情スコアを得るわけだが，先ほどの negation の例も
あるから，顔文字がある場合にはそちらのスコアのみを使う

* step 4
sentiment score of segment

segment s に
  v コの顔文字 (e[1] .. e[v])
  t コの sentiment-carrying or modifying word (w[1] .. w[t])
があるとする

s の感情スコアを次のようにする

sent(s) = 
  if v > 0 then
    (sum e[i]) / v
  else
    (sum w[i]) / t

最後に，document に対する感情スコア

d = s[1] .. s[p]

sent(d) =
  (sum_i=1^p (if v[i] > 0 then (sum e[j]) else (sum w[j])))
  / (sum (if v[i] > 0 then v[i] else t[i]))

class(d) = if sent d < 0 then -1 else +1

# 評価
2080 の Dutch tweets and forum messages (Google discussion)
について
class d
を人手でつけて
比べた
87.5%正しかった
